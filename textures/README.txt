=== PAQUETE DE TEXTURAS DE MYMINECRAFT ===

¡Has activado el soporte para paquetes de texturas personalizados!

En esta carpeta encontrarás los archivos PNG para cada cara o bloque del juego.
Cada imagen mide 16x16 píxeles para mantener el estilo retro y pixel-art clásico.

CÓMO PERSONALIZARLAS:
1. Abre cualquiera de las imágenes PNG con un editor de imágenes o pixel-art (como Aseprite, Photoshop, Paint.NET, GIMP, o incluso Paint).
2. Modifica el dibujo o los colores.
3. Guarda el archivo con el mismo nombre en esta carpeta.
4. Recarga la página en tu navegador (http://localhost:8080) para ver tus nuevas texturas aplicadas automáticamente en tiempo real.

ARCHIVOS DISPONIBLES:
- grass_top.png   -> Parte superior del bloque de césped
- grass_side.png  -> Caras laterales del bloque de césped
- dirt.png        -> Bloque de tierra (y parte inferior del césped/nieve)
- stone.png       -> Bloque de piedra
- wood_top.png    -> Parte superior e inferior de la madera (tronco)
- wood_side.png   -> Caras laterales de la madera (corteza)
- leaves.png      -> Bloque de hojas (soporta transparencias)
- sand.png        -> Bloque de arena
- glass.png       -> Bloque de vidrio (soporta transparencias)
- brick.png       -> Bloque de ladrillos
- snow_top.png    -> Parte superior de la nieve
- snow_side.png   -> Caras laterales de la nieve
- bedrock.png     -> Bloque de bedrock indestructible
